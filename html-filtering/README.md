html-filtering.py
=================

a python program that removes HTML tags from an input file or url.

usage
-----

`python3 html-filtering.py test.html`

or

`python3 html-filtering.py https://orf.at`

output for test.html
--------------------

file detected, reading...
filtering html tags...
result:


  
    
    
    

    
    

    Hello, world!
  
  
    Hello, world!
